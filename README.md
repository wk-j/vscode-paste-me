## Paste Me

Generate list of quick paste items from command line.

![](https://github.com/wk-j/vscode-paste-me/raw/master/images/screen.png)

## Settings

```json
{
    "pasteMe": {
        "commands": [
            "h-extensions",
            "h-reference \"${rootPath}\" \"${fileName}\""
        ]
    }
}
```

## Usage

- Command + Shift + J
- Select item
